# Как пользоваться ботом

Эта страница описывает пользовательские сценарии v1 без технических деталей разработки и деплоя.

## Что умеет бот

Сейчас бот помогает:

- зарегистрироваться через Telegram;
- посмотреть свой профиль с ролями, компетенциями и баллами;
- запросить роль `Student`, `Mentor` или `Admin`;
- посмотреть недельный лидерборд;
- посмотреть роли и компетенции пользователей;
- администраторам управлять баллами, ролями, компетенциями и рассылками.

## Регистрация

Чтобы начать работу:

1. Откройте личный чат с ботом.
2. Отправьте `/start`.
3. Поделитесь своим номером телефона через кнопку.
4. Заполните имя, фамилию и отчество.
5. Выберите компетенции или пропустите этот шаг.

Компетенции на этапе регистрации нужны как стартовая информация о том, что вам уже знакомо или интересно. Если пока не уверены, шаг можно пропустить.

## Профиль

Команда `/profile` показывает:

- академический уровень и академические баллы;
- репутационный уровень и репутационные баллы;
- роли;
- компетенции.

Сейчас профиль нельзя полностью редактировать самостоятельно через пользовательские команды. Для изменений, которые уже доступны в системе, обычно нужен администраторский сценарий.

## Роли и права

- `Student` — базовая учебная роль пользователя.
- `Mentor` — роль наставника с дополнительными возможностями в рабочих сценариях платформы.
- `Admin` — роль для административных команд и обработки role request.

Если нужной роли у вас ещё нет, можно отправить запрос через `/role_request <Student|Mentor|Admin>`. Решение по запросу принимает администратор.

## Баллы

В боте есть два типа баллов:

- `academic` — отражают учебный прогресс;
- `reputation` — отражают вклад, помощь и полезность для сообщества.

Пользователь может видеть свои баллы и уровни в `/profile`, но начислять и изменять баллы могут только пользователи с административным доступом.

## Команды

### Пользовательские команды

- `/start` — начать регистрацию или открыть профиль, если вы уже зарегистрированы;
- `/help` — краткая подсказка по командам;
- `/info` — краткая информация о проекте;
- `/profile` — ваш профиль;
- `/leaderboard` — недельный лидерборд;
- `/showroles [@user|id|reply]` — показать роли пользователя;
- `/showcompetences [@user|id|reply]` — показать компетенции пользователя.

### Администраторские команды

- `/academic_points @user <число> "причина"` — изменить academic-баллы;
- `/reputation_points @user <число> "причина"` — изменить reputation-баллы;
- `/addrole @user <Student|Mentor|Admin> "причина"` — выдать роль;
- `/removerole @user <Student|Mentor|Admin> "причина"` — снять роль;
- `/addcompetence @user <список>` — добавить компетенции;
- `/removecompetence @user <список>` — удалить компетенции;
- `/broadcast ...` — отправить рассылку.

## Known Limitations

- Самостоятельное редактирование профиля пользователем остаётся вне v1 и частично закрывается только через администраторские сценарии.
- Полноценный audit log для пользовательских и административных действий относится к post-v1.
- MCP-интеграции не входят в v1 пользовательского релиза.
